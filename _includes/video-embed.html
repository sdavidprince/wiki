{% comment %} jekyll-embedded-video · Nathan Lam, Josh Fogg · 2017-2021 Embedded video player for Jekyll which works which a whole host of different websites without a plugin. It takes the following two required parameters site: the service hosting the original video id: that service's ID for the original video and any combination of the following optional parameters alt: video title to use as title and alt text autoplay: bool, whether video starts playing automatically fullscreen: bool, whether player is allowed to go full screen width: width of the embedded window, in pixels height: height of the embedded window, in pixels and then returns a responsive iframe to embed that video in a page. See the README for more explanation and example usage. {% endcomment %} {% comment %} Set aliases and defaults for input variables.
